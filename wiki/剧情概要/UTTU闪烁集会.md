# UTTU 闪烁集会

> "闪烁集会是由 UTTU 杂志社发起的读书会活动，是近几年在神秘学界相当受追捧的集体活动。它的举办地点变幻莫测，主题多样神秘，唯一不变的就是书目有趣，奖赏丰厚。"  
> ——梅兰妮，《昨日金杯》

## 概述

**闪烁集会**是[[世界观/UTTU|《UTTU》杂志社]]近年发起的**读书会式集体活动**，神秘学界普遍追捧。剧情上以**零碎短篇**形式呈现，每期一个主题、若干主角围绕一本杂志的内容展开闲谈。

游戏内通常以**玩法活动**形态出现，文本量轻，作为常规活动外的零碎补完。

## 闪烁集会运作机制（首期《昨日金杯》揭示）

| 环节 | 内容 |
|---|---|
| **报童** | 戴报童帽的孩子们在街角散发杂志，"忽而出现，忽而又不见踪影" |
| **闪烁侧刊** | 报童赠送的小册子，单本不构成完整内容 |
| **UTTU 零售** | 集会的临时分发点，由报童在场即开 |
| **召唤仪式** | 收集足够数量的闪烁侧刊后，把它们摆成阵——"一道光"召出**UTTU 超级市场** |
| **UTTU 超级市场** | 由"一张嘴+一群蜘蛛+一道光"显形而开启的神秘空间，内部可阅读**闪烁总刊**（每期完整内容） |
| **奖赏** | 物质方面"奖赏丰厚"（首期未具体揭示） |

> 报童团体的出没形式神秘——成群出现又瞬间消失，"几乎是一眨眼"。与 UTTU 一贯的[[世界观/UTTU#杂志的神秘学属性|"配送方式神秘学化"]]一脉相承。

## 第一期：昨日金杯

### 主题与基调

围绕**世界杯历史**与"足球之父"**儒勒·雷米**展开。借小狗皮克勒斯阅读杂志的视角，对足球与战争的关系、形式与精神的关系做出短小思辨。

### 出场人物

- **[[角色/圣洛夫基金会/皮克勒斯|皮克勒斯]]**（小狗）：起初为追温迪丢掉的球而误入；阅读杂志后改变了对"足球只是吵闹娱乐"的认知。每段吠声后均由 "Doggie" 提示其心境
- **[[角色/独立与其他/挖掘艺术|挖掘艺术]]**：闯入"闪烁集会"的不速之客，热衷于把任何东西诠释为艺术；最后混入庆祝人流
- **[[角色/圣洛夫基金会/梅兰妮|梅兰妮]] / 艾西**：以怪盗"艾西"身份多次参加闪烁集会，是本场流程的指导者。她揭示了"召唤 UTTU 超级市场"的玩法
- **报童团**：UTTU 派发员，瞬现瞬消

### 剧情节点

1. **偶遇 UTTU 零售**：皮克勒斯在公园追球时撞见挖掘艺术；报童群从人群中钻出，赠予一本闪烁侧刊后瞬间消失
2. **梅兰妮入场**：以怪盗艾西身份追集刊（被艾西老师评 60 分）；与皮克勒斯/挖掘艺术合并；解释闪烁集会规则
3. **集齐侧刊**：皮克勒斯凭嗅觉穿过人群迅速找到下一处 UTTU 零售（"第 29 位访问者"），三人完成集刊
4. **召唤超级市场**：草地上摆放杂志阵，光与蜘蛛涌现，UTTU 超级市场之门开启
5. **阅读闪烁总刊**：小狗皮克勒斯翻动书页——内容为儒勒·雷米创立世界杯的历史，足球如何缓和国际矛盾，世界杯曾因战争停办三届等
6. **挖掘艺术的演讲**：从"梅开二度""绝杀球"等术语跳到"艺术=寄托于形式上的精神"——"过去的只是比赛这个形式，而不是精神本身！足球精神永存！艺术永存！"
7. **梅兰妮的内心独白**：从"过去的只有形式"中读出对**拉米雷斯**（她已故的亲人/同伴）的怀念——"拉米雷斯从未离开过我"
8. **预言性收尾**：梅兰妮望向街角："没准再过一段时日，连'乌卢鲁'的圣火都能重新燃起呢。"（呼应[[剧情概要/活动章节/04_复兴！乌卢鲁运动会|乌卢鲁运动会]]）

### 主题

- **形式 vs 精神**：挖掘艺术的"艺术=精神"论 + 梅兰妮的"拉米雷斯从未离开"——给悼亡情绪一个温和出口
- **战争与体育**：世界杯三度停办；儒勒·雷米作为"利用足球缓和国际矛盾的人"——延续游戏一贯的"暴雨/战争与人的对抗"母题
