# 世纪末尺度 / The Temporal Scale

> “它带走了地球上绝大多数人口，其中也包括我的亲人。世界将我们带回了1996年。”——[[#蓝图（艾达）|蓝图]]，TTS-01

> “在我看来，诺瓦米奇的重建并非勇敢的象征，它只是一个判断——我们是否还愿意在不确定中行动，而不是等世界推着我们做决定。”——[[#奠基石（斯塔夏）|奠基石]]，TTS-18 录像带

## 概述

**世纪末尺度**是 v3.8 版本登场、共 **TTS-01 ~ TTS-22** 22 节的主线性质章节，时序上是一段**前传/番外**——讲述基金会在**第一次"时空灾害"**（1999-12-31，将人类回退至 1996 年）之后，于东欧小城**[[地点/诺瓦米奇]]**建立**首个"不可回溯地"庇护设施**的全过程。本章承担多项重大世界观补完：

- **"暴雨"命名史**：原名"时空灾害"，在 TTS-22 中被官方正式更名为"暴雨"
- **"不可回溯地"概念**确立：免疫时空灾害的特殊区域（详见[[世界观/暴雨免疫区]]）
- **回声石**与诺瓦米奇穹顶机制：基金会第一次以**建筑**而非神秘术建立稳定避雨点
- **"新办事处建设"项目**：基金会未来在世界各地建立避雨办事处/分部的范式起点
- 基金会内部两条路线（**信息获取派**：[[#肯纳|肯纳]] vs **安全收敛派**：[[角色/圣洛夫基金会/伊里内伊|伊里内伊]]）的最早一次正面交锋
- 副会长**[[角色/圣洛夫基金会/康斯坦丁|康斯坦丁]]**对"非必要项目"的关键松手——成为其后整个避雨办事处网络得以铺开的源头

> **性质归属**：本章编号格式（TTS-）与 10TH/11TH/13TH 同构，体量 22 节，承担核心世界观揭示，故归类为**主线性质章节**，归档于 `raw/主线故事/`。但它**不推进主线人物当下剧情**（维尔汀、十四行诗、司辰小队均未登场），更接近"基金会内部往事"前传——与"77 号往事"（v2.1）相似但规模更大。

## 时序锚

|---|---|

| 1996 年初 | 拉普拉斯[[角色/拉普拉斯科算中心/露西|露西]]提出"不可回溯地"候选地图；东欧诺瓦米奇标为唯一暗红点 |
| **1996-02-18** | 蓝图抵达诺瓦米奇，与本地出身的建筑师奠基石会合（TTS-02） |
| 1996-02-23 | 穹顶项目正式开工，居民临时工队组建（TTS-08） |
| 1996-03-10 | 第十六天，西侧 + 居民区回声石拆除完成（TTS-11） |
| 1996-06-12 | 项目暂停第七天，奠基石被噩梦袭击后采风（TTS-15） |
| 1996-07-01 | 拉普拉斯发出"时空灾害可能再来"预警，重建获康斯坦丁授意重启（TTS-19） |
| 1996-07-15 | 团队扩招、施工重启（TTS-20） |
| **穹顶落成日** | **第二次"时空灾害"**（即后来命名的暴雨）降临；回声谣以自身完成穹顶；穹顶下数十人安然存续；基金会绝大多数分部成员仍未能免于回溯 |
| **1987-04-03** | 第二次暴雨**回退后**的现世时点；奠基石回访诺瓦米奇办事处，"时空灾害"已被正式更名为"暴雨"（TTS-22） |

> **暴雨纪年与本章**：v3.8 给出的"第一次时空灾害=1999-12-31"是**基金会内部命名史的第一次**，并非历史上的首次暴雨——这与[[角色/拉普拉斯科算中心/乌尔里希]]在 10TH 提出的"历史上所有'暴雨'皆为失败洪水仪式残余"假说自洽。即从基金会观测视角，1999-12-31 为命名零点；从历史回顾视角，暴雨已发生过多次（参见[[世界观/暴雨时间线]]）。

## 登场人物速查

### 基金会方
- **[[#奠基石（斯塔夏）|奠基石 / Cornerstone]]**：基金会专职建筑师，诺瓦米奇本地出身（8 岁随父母移居海外），项目设计与主导者。乳名"斯塔夏"
- **[[#蓝图（艾达）|蓝图 / Grundriss]]**：基金会监理工程师，伊里内伊下属，妹妹汉娜在第一次时空灾害中消失。"Grundriss"即德语"平面图/底图"
- **[[角色/圣洛夫基金会/柏林以东|柏林以东 / Bkornblume]]**：先遣调查员兼监听员；与隐秘上级保持联络
- **[[#肯纳|肯纳 / Kenna]]**：基金会工程部红发女士，"新办事处建设"项目对外推动者，原格拉斯哥办事处调任
- **[[角色/圣洛夫基金会/伊里内伊|伊里内伊 / Irinei]]**：基金会委员，"新办事处建设"项目内部负责人；安全收敛派
- **[[角色/圣洛夫基金会/康斯坦丁|康斯坦丁 / Constantine]]**：副会长，关键时刻为重启项目松手
- **[[角色/拉普拉斯科算中心/露西|露西]]**：拉普拉斯纽约第三分部调来的代理负责人，主持"时空灾害"研究汇报
- **希曼**：基金会高层职员，与露西、康斯坦丁配合（暂无独立词条）

### 诺瓦米奇本地
- **[[#回声谣 / Everecho|回声谣 / Everecho]]**：诺瓦米奇地下千年积淀的回声生命体；最终以自身完成穹顶
- **小洪扎 / Honza**：回声谣手中的木偶伙伴
- **奥尔加**：本地"话事人"，老酒保兼酒馆"老橡树"实际经营者；奠基石幼年记忆中的故人
- **亚肖**：流浪汉、临时工头，灾后留在诺瓦米奇"不知道该往哪儿去"
- **卢齐安**：苍白男孩，喜爱看电视；途中牺牲，化为穹顶能量
- **罗莎**：奠基石幼年的导师/邻人（已亡），教过她"寂静的声音"
- **马尔钦、希蒙**：临时工人

## 分节剧情

### 起势（TTS-01 ~ TTS-03）：会议与抵达

- **TTS-01**：1996 年初基金会大会。露西公布"不可回溯地"候选图，唯一已派员的暗红点是东欧诺瓦米奇——本地出身的[[角色/圣洛夫基金会/柏林以东]]已先行勘察。肯纳与伊里内伊以"新办事处建设"项目共同负责人身份，要求基金会在"不可回溯地"上建立长期办事处。伊里内伊指派蓝图为"建筑师奠基石的助理"前往。
- **TTS-02**：1996-02-18，蓝图独自抵达诺瓦米奇车站。柏林以东远程指路时通讯被建筑切断。一名年轻流浪汉（亚肖）出现。雨中，奠基石撑起白色立面赶到，二人会合。
- **TTS-03**：两人步入"闹鬼圣地"。奠基石将耳朵贴上墙面，墙体"温暖"，从中听见埃米尔与罗莎童年的呼唤声。手偶小洪扎与"鬼魂"回声谣登场；柏林以东端枪驱散，自报家门。

### 发现（TTS-04 ~ TTS-07）：回声石与人

- **TTS-04**：柏林以东介绍这栋建筑分为西侧（"鬼魂"游荡区）与东侧（居民暂住区），墙中含有"回声石"。建筑师在中庭原穹顶残骸下听见熟悉的童年呼唤，发现穹顶曾是等边三角形支撑下的圆顶结构。卢齐安登场。
- **TTS-05**：奠基石提出"复现穹顶"。男孩卢齐安从冷漠到答应带她见话事人奥尔加。
- **TTS-06**：在酒馆"老橡树"，奠基石为居民们逐一画"愿望中的房子"草图（卢齐安要"流水别墅"、柏林以东要"种矢车菊的安全屋"）。深夜，老酒保自报姓名——她正是奥尔加。
- **TTS-07**：奥尔加点头，居民动员到位。

### 施工（TTS-08 ~ TTS-11）：从迟到的工人到拆除居民区

- **TTS-08**：1996-02-23 开工，所有人迟到。蓝图严苛立规，发安全帽与施工手册；同时奠基石遭到一道"阴影"的偷袭——草图、铅笔反复失窃。
- **TTS-09**：柏林以东与隐秘上级通话（**"原则上你应当置身事外，但已经到了这个阶段……你可以试着提供一些帮助"**——上级身份未明，是本章重大悬念）。蓝图磨合下，工人渐入轨道；与此同时，回声谣与小洪扎偷走奠基石的祖传橡皮"作为左膀右臂", 小洪扎几乎瓦解，被赶来的奠基石救下。
- **TTS-10**：回声谣与奠基石初步对话。
- **TTS-11**：1996-03-10 第十六天，墙体回声石与居民床头石全部拆除归集。奠基石以"侧光"方案（提升屋盖留出连续高窗）解决厨房采光，灵感来自圣索菲亚大教堂与童年画作。

### 危机（TTS-12 ~ TTS-17）：回声谣的来历与卢齐安牺牲

- **TTS-12**：回声谣**起源回溯**——她是诺瓦米奇土地上千百年来积淀的歌声集合体，"被写在士兵的信纸上，被归家的老人在田埂上哼唱"。奠基石幼年时罗莎与"埃米尔"曾教她听"寂静的声音"——金色光带，那正是回声谣的雏形。
- **TTS-13~14**：与隐秘对手的对峙。
- **TTS-15**：1996-06-12，项目暂停第七天。蓝图向柏林以东讲祖父——**包豪斯最后一届学生**的故事（1933 年学校关闭，那个"群星闪耀年代"的尾声）。山坡采风途中，奥尔加讲城市重建史，奠基石**忽然看见整座城市的"血管"**——条带式秩序，砖块"想成为什么"的问题终于有了答案：回声石的排布应当贴合自然律。同节卢齐安向回声谣点破：**她其实并不想阻止他们**。
- **TTS-16**：模型成功发光（即"圣光"）；回声石复苏；众人发觉**卢齐安从一开始就是"鬼魂"**，最终化作光，"回到了光里"。回声谣承认她身边的回声石"是我们的记忆之所"。
- **TTS-17**：1996 年 6 月 28 日，项目暂停第 23 天。柏林以东启程带模型回总部去说服基金会；小洪扎重伤失语（卢齐安牺牲的余波之一）。回声谣对奠基石说**"卢齐安在光里、在树叶里、在咖啡里、也在这里"**，达成内心和解。

### 转机与重启（TTS-18 ~ TTS-20）：康斯坦丁松手

- **TTS-18**：肯纳带柏林以东闯进康斯坦丁办公室——拉普拉斯刚发来报告：**"时空灾害"有可能再度降临**。肯纳的核心论点："我们正在把生存当作目标，而不是前提"。康斯坦丁初次拒绝。她离开后独自播放奠基石录制的录像带（**完整文本见 TTS-18，是本章主题宣言**），翻阅蓝图项目日记。最终召柏林以东回办公室。
- **TTS-19**：1996-07-01 重启。奠基石手绘海报、霓虹灯招牌登街宣传，电视台《公共之声》报道——意外成为基金会内部围观新闻。康斯坦丁默许扩招。
- **TTS-20**：1996-07-15 重启施工。穹顶落成日，最后一块回声石升至最高点时**第二次时空灾害降临**，仪器同步报警。小洪扎"开口"——念出诗句，化作巨兽。回声谣捡起破碎的手偶，目睹卢齐安等"回声"现身。

### 决断与告别（TTS-21 ~ TTS-22）：穹顶之下

- **TTS-21**：医院涌现"电线人"症候（这是第二次暴雨的症候群——血管变电线，呼应[[世界观/暴雨#暴雨症候群|第二次暴雨症候]]）。蓝图、奠基石、柏林以东三人主动留下与居民共渡。回声石数量不足以让穹顶全覆盖，外来涌入的家眷无处安置。**回声谣飞出穹顶**，以自身"加入回声"完成穹顶——金色光带从天倾泻而下，将所有人安然庇护，雨被隔绝在三角穹顶之外。回声谣留下的最后一句："**愿你们都深信明天。**"
- **TTS-22**：基金会内部"暴雨"应急系统会议——"时空灾害"已正式更名为**"暴雨"**。奠基石被派往普列谢茨克评估拉普拉斯办公室作为新"不可回溯地"的可行性。康斯坦丁与肯纳之间路线分歧公开化（肯纳："那不是'意外'，而是'奇迹'。在我看来，一个收敛的系统中很难诞生出'奇迹'。"）。**一年后（1987-04-03）**，蓝图与奠基石回访诺瓦米奇办事处；奥尔加仍在；中庭独留奠基石听见**罗莎、奥尔加、回声谣的声音重新显现**。镜头最后定格于雏菊花丛中的小洪扎——**"过去不曾远离，歌声也将永远继续。如影随形，永不消失——你猜对了，正巧是我。"**

## 主题与意义

1. **"时空灾害"→"暴雨"命名史**：v3.8 给出了基金会从"灾害命名 → 现象常态化承认"的关键过渡——TTS-22 之前文本一直称"时空灾害"，回访段（一年后）才用"暴雨"。这与全作其他章节使用"暴雨"为常用名形成一致解释链。
2. **避雨办事处网络的起点**：诺瓦米奇办事处是基金会未来"在世界各地寻找不可回溯地并建立避难所、仓库、实验室"的第一块砖（详见[[世界观/暴雨免疫区]]）。
3. **路线之争的早期形态**：肯纳（信息获取/愿押注未来）vs 伊里内伊（安全收敛/现实主义）——是后来主线第十一/十二章鸽派-鹰派分裂的雏形。
4. **回声石的本质**：穹顶之所以能形成"不可回溯地"，是因为石中栖居着"过去的回声"——亡者的、消失者的、被遗忘者的痕迹。这构成了一条独立于"非对称核素 R"的免疫机制路径（详见[[世界观/暴雨免疫区]]补充）。
5. **柏林以东的隐秘上级**：TTS-09 那位指示"已经到了这个阶段……你可以试着提供一些帮助"的对象身份是悬念。结合柏林以东已知代号 SCH KA/7 与其外勤身份，可能与基金会内一支专门处理"不可回溯地"事项的暗线相关，**待后续版本揭示**。
