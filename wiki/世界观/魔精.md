# 魔精 (Carbuncle)

> "它们不再是令人恐慌的怪物，而是我们另一种模样的朋友。"——新巴别塔

## 概述

**魔精**（又称**卡邦克鲁 / Carbuncle**）是《重返未来：1999》世界观中常见的**神秘学生物**统称——既指那些标志性的"毛绒绒小家伙"卡邦克鲁，也广义涵盖一系列神秘学动物（鸟目魔精、飞行魔精、拟人魔精等）。它们栖息于荒原与隐秘角落，过去被视为"令人恐慌的怪物"；经[[角色/独立与其他/新巴别塔]]在**不休荒原·魔精公司**中以神秘术驯化后，转化为可参与生产、可与人共处的"魔精朋友"。

## 物种谱系

| 类型 | 特征 | 代表 / 出处 |
|---|---|---|
| **卡邦克鲁** | 小型、毛绒绒、扎堆喜欢；情绪丰富、有大眼睛；会偷东西藏到帽子里；闹起来"魔精暴动" | 主要类型；TTS-04/05/15 与新巴别塔魔精公司中的"小家伙们"皆此 |
| **粘稠魔精** | 粘稠状 | 不休荒原·魔精公司 |
| **刺挠魔精** | 长刺/刺挠 | 不休荒原·魔精公司 |
| **拟人魔精** | 形似人类 | 13TH-23 兔毛手袋抱怨"有什么拟人魔精在这儿大喊大叫" |
| **鸟目魔精**（鸟类魔精） | 大型，鸟形，可被人骑乘 | 梅蕾尔《鸟目魔精习性与饲养指南》（沙盘解构·他者的悲哀） |
| ↳ **尘酿鹰** | 莓果园栖息；雄壮身躯+强翅膀；可骑乘翱翔；**数量已经很少** | 同上 |
| ↳ **狮鹫** | 大型鸟类魔精；推测与尘酿鹰是亲戚 | 同上 |
| **飞行魔精** | 群居飞行型 | 第十三章及《应急预案手册》 |
| ↳ **奥利图欧群** | 重塑之手**大规模饲养**用于战场 | 沙盘解构·他者的悲哀（1921 年战场空战） |

> **命名约定**：剧情中"魔精"是大类统称，"卡邦克鲁"特指那种圆滚滚的毛球子物种。两词常混用，但在新巴别塔魔精公司剧情中可见区分（"我们的卡邦克鲁"vs"另一只刺挠的魔精朋友"）。

## 行为与习性

- **群居扎堆**：常成群结伴，"齐齐整整地盘踞"在墙缝、石缝（TTS-04 奠基石搬开石头时一窝魔精数双眼齐刷刷看向她）
- **情绪外显**：受惊→"惊叫着四散奔逃"；好奇→歪头凑近；不满→闹腾打架
- **偷东西/恶作剧**：把老马尔钦的安全帽戴在自己头上（TTS-15）、从打印机上叼下订单（魔精公司剧情）、咬蹭羽毛
- **朋友圈**：库珀花环观察发现"它们总是在为自己挑选朋友"——朋友间非常友好，但某些卡邦克鲁碰面就打架（库珀花环·荒原对话）
- **争夺领地与繁育本能**：X 提醒新巴别塔："争夺领地与资源，结成聚落……接着便是无止境的繁育本能"——必须控制数量
- **昼夜规律**：从门的台词推测部分卡邦克鲁是**晨型生物**——"森林里的卡邦克鲁开始苏醒"
- **饮食偏好**：会啃手套、试剂、门板、沥青水泥、草坪等等；也喜欢吃糖（见下文"扭扭糖球机"）
- **语言**：未知；数羊羔尝试采访箱中魔精，记录"魔精和外面的并没有什么区别……如果我能听懂它们的语言就好了"

## 新巴别塔魔精公司

详见 raw/H5活动/不休荒原·魔精公司·剧情。

[[角色/独立与其他/新巴别塔]]在不休荒原（维尔汀的箱中世界）创办的公司。

**触发**：箱中荒原原本生活着新巴别塔带去的魔精（她称它们"孩子们"——洛蕾塔、阿诺德、希瑟、**帕梅拉**等）。某次魔精们突然群体躁动、闹作一团；新巴别塔以**克鲁山德家神秘术**（钟楼洒下的"微尘雨"）安抚平息后，向维尔汀提出商业方案。

**核心运作**：
- **驯化教学**：新巴别塔用神秘术影响魔精行为模式，引导它们从事生产
- **业务范围**：
  - 协助发放传单（沙丝绒的需求）
  - 初级魔药试验
  - 检测土壤
  - 简单松土
  - 开垦新荒原
- **派遣机制**：箱中训练好的卡邦克鲁可被外派到不同岗位，由"魔精朋友"身份完成工作
- **舆论铺垫**：[[角色/UTTU/数羊羔]]报道"荒原上的魔精"后，**经培育的卡邦克鲁引起神秘学界广泛关注**；"新巴别塔今晚秀"宣传节目同步上线

**新巴别塔变异魔精**（剧情自述）：箱中环境催生的变异种，与外界自然魔精有性格差异；新巴别塔以此为素材撰写《神秘学家与"新巴别塔"变异魔精的二三事》。

### X × 洛伦兹蝴蝶的"扭扭糖球机"

为解决魔精数量/繁育/打架问题，[[角色/洛伦兹研究所/X|X]] 与新巴别塔达成合作：
- 新巴别塔提供精力旺盛的魔精志愿者作 X 研究魔精生理与行为
- X 提供 **"扭扭糖球机"**——摇动机器生产风味独特的魔精糖果，**既能控制数量、提升质量，又实现规模化/自动化生产**；糖果将不同种类魔精吸引到主人身边，转移它们的好斗注意力

X 透露这项研究中"某位蝴蝶女士提供了充分协助——知识、灵感，以及恶趣味"——即 [[角色/洛伦兹研究所/洛伦兹蝴蝶|洛伦兹蝴蝶]] 在魔精学上的另一面。

## 现实参照

- **Carbuncle / 卡邦克鲁**：南美/西班牙殖民地传说中一种身怀珠宝、能在夜里发光的小动物，常与圭亚那、巴塔哥尼亚的丛林相关；在 JRPG（特别是 Final Fantasy 系列）中以"喷气宝石、额头发光的萌系幻兽"形象广为人知。本作沿用其"萌系小生物+珠宝光泽"基调
- **物种谱系命名**沿用神话/猎奇生物学传统：尘酿鹰、狮鹫等取材自欧洲神秘动物图鉴
