# “这座城市是我们的”
人人都知道该避开什么。
城与“城”
“暴雨”之下，夜巡特遣管理局几乎以同一种方式出现在每一座“重要的城市”——起初是一间不起眼的办公室，随后是几名执夜人，再往后，人们便逐渐习惯它的存在，仿佛它原本就该存在于此。

在官方记录中，夜巡特遣管理局的职责被界定为调查与神秘学相关的疑难案件。它与政府机构各司其职，双方分工明确，互不干涉，却始终保持稳定的联系。

与此同时，地下帮派进入了一个新的阶段，被报纸称作“现代化犯罪”。“灰车党”在这一时期逐渐有了辨识度——灰色汽车、相对统一的着装，以及更有组织的抢劫与暴力行为，使他们频繁出现在各大报刊的社会版面。新型作案方式被反复拆解、讨论，但这些分析并未削弱其犯罪属性，相反，它们让人逐渐意识到，暴力与掠夺正在以更系统、更高效的形式发生。犯罪不再只是治安部门需要应付的事件，而是成为整个社会不得不直面的现实问题。

当然，灰车党并不足以撼动整座城市的秩序。本地警察依旧清楚自身的权责所在，夜巡特遣管理局亦明白如何避免提出超出管理范畴的质询。地下势力与官方系统共享着同一张城市地图，且各自保留对其的解释权。

值得注意的是，这种看似和谐的共存并不能使冲突完全消失，它们只是被限制在了一定范围内。警察、执夜人与灰色车辆穿梭于同一条街道，彼此知晓对方的存在，他们心知肚明——当太阳升起，街道将恢复原状，眼前的一切又会再度融入日光，化作城市的影子。

# 尘埃之下
我们所急需的艺术文化。
壁画复兴运动纪实其一
二十世纪初的墨西哥，几乎没有哪种艺术形式比壁画更深入日常——它们的尺寸远超私人收藏，它们被固定在楼梯、庭院与长廊一侧。也正因此，壁画运动从一开始便引发了持续不断的讨论——关于艺术本身，关于该由谁决定艺术的呈现方式，以及这些作品究竟能在公共空间内停留多久。

曾经，审美的方向被政府牢牢把控。直到旧秩序崩塌，被禁锢的传统文化才重获新生。公共建筑成为画布，图像便自然而然地承担起叙述与表达的职责。大幅壁画占据了走廊与庭院，在功能层面，它们越来越接近于路标或门牌，而非供人观赏的绘画。人们总能在不同的画面中看到相同的姿态——抬头、劳作与行进。观看它们，理解它们试图传递与表达的内容，也逐渐成为墨西哥人生活的一部分。在艺术文化的引领下，墨西哥人得以建立全新的身份认同感。

在技法上，这些作品多以湿壁画的形式完成。画家必须在灰泥尚未干透前作画，颜料随之渗入墙体，一旦凝固，几乎无法更改。随着时间推移，颜色逐渐暗淡，画面的轮廓逐渐被尘埃掩盖。无数人经过、抬头、离开，而墙却始终存在于此，它始终保留着被使用过的痕迹。

也许，真正重要的并非壁画的形式与革新的技法，而是附着其上的情感与记忆。

# “清洁注意事项”
第一条，别把龙舌兰当菠萝吃。

“必要”的除灵指南
1.龙舌兰

使用注意事项：别被大叶子割伤了手。
现实用途：常用于酿酒、制药与各种仪式，其汁液发酵后制成的普逵酒被墨西哥人视作“土地的体液”。

# 古老日历
我们会见到第六个太阳么？

太阳石的故事
阿兹特克太阳石常被称作“阿兹特克日历”。它其实更像一张刻在玄武岩上的宇宙结构图，抑或阿兹特克人所信奉的世界运行指南。

这块石头出土于18世纪的墨西哥城，现存国家人类学博物馆，直径超过3.5米，重量在20吨以上。太阳石的中心是太阳神托纳蒂乌，祂神情严肃，伸出的舌头和燧石刀十分相似——如果不把最宝贵的事物，也就是血献给太阳，那么太阳不会赐予你任何祝福。

围绕祂展开的同心圆，记录着此前已经沉没的四个“太阳”——一个被美洲虎吞噬，一个被飓风撕碎，一个被火焰焚毁，最后一个被洪水淹没。每个“太阳”的结局都是末日级别的灾难——没有任何一个“太阳”能稳定地延续下来。

再往外，你会瞧见宗教日符号、暗示昼夜交替循环的图形，以及代表天体的颜色……它们层层展开，如同一台被拆解后再摊开展示的宇宙机器——精密、古朴、有序。它们代表了阿兹特克人眼中的世界及他们对宇宙的理解——宇宙重启，万物新生，人类不过是其中相当细小的一粒微尘。

# 必然一步
我们相聚，我们分离，皆因同一个词语。
壁画与转折
这片土地上生长着以壁画诉说的传统。在革命之后，年轻的政府向艺术家们伸出了橄榄枝。新生的墨西哥有太多故事需要表达，需要塑造，公共建筑上的壁画是绝好的载体。1920年代，一场名为墨西哥壁画运动的浪潮席卷了此处。

瞧，这是一种传统而崭新的艺术。它不再待在拍卖会、画廊或富人的收藏室，它在巨大的、向民众开放的墙面上铺展开来。艺术家在墙壁上讲述着——关于何为正义，何为自由，何为这个国家的历史与记忆。请想象吧！一个新的国家，一片新的画布，许多、许多新的故事！从诞生之日起，这些新壁画就是同政治息息相关的。画笔是画家们的喉咙，他们为理想发声，为美发声，为全世界受苦的人们发声……

声音，声音是难以统一的，是会变得刺耳的。

二十世纪进入了第三个十年，画家与画家、画家与国家的声音逐渐出现了参差。该描绘苦难还是荣光？该歌颂谁？该选择哪一条道路？里维拉、奥罗斯科和西凯罗斯——三个象征希望的名字，我们的“墨西哥壁画三杰”——对国家与绘画的前途有着不同的见解。

纷争越来越大。在小小的“午夜厨房”里，年轻的画家们亦听到了这场“世纪之战”。
