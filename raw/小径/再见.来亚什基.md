# 一个承诺

明亮的雪地
    再好好瞧瞧广场。它变得不多——中间有一只轮廓柔和的北极熊，孩子们伸手去摸它的脚爪。雪覆盖了熊的头顶，使它显现出一种与愉快气氛不符的沉思感。在整片地面上，花朵几乎盖去了所有石头和白雪，熊安静地生活在一片灿烂的明黄色中。

    这是来亚什基最重要的场所之一，也许因为小镇的厨房里挤不下那么多人。许多公开决议都在这儿举行，此后也将继续下去。不久前，人们决定接受另一份邀请，让孩子们走到外边看看。沿着那条道路——被阳光映亮的道路，树上缠绕的红布条在风里上下飘动。

    人们在这里找到了自己熟悉的影像，一些过往生活的痕迹。尽管踏上这片土地的人很少谈论起他们的家乡。来亚什基应该是这样的，是他们曾经期望能在故土看到但从未实现的样子。就像穿越寒流的鲁萨尔卡远远望见的地平线那样，像四处奔波的理线师描绘的图景那样，像造访的基金会成员不复存在的故乡那样，像上一代人向下一代许诺的那样。

    理想越难以实现，就越伟大，因不可实现而不可摧毁。

    生活中还有更多事等着人们去做。

雪与其他可能
对小镇来说，这是个熟悉的身影。对来者自己而言，一切都如此陌生。

她习惯了在雪中穿行。莫斯科街头飞舞的杨絮会令她想起来亚什基，它与雪很相近，却不会使人脸颊通红，手足冰冷，不会在皮肤上化开，也无法遮掩任何事物。商店里能买到罐头，甚至香肠和新鲜的肉，想置换些日用品也不必掰着指头数时间……第一次在新搬的屋子里炖汤的时候，她感觉这不真实。另一件让她印象颇深的事是刚下火车时，她吓得攥紧了孩子的手——在小镇之外的地方，在莫斯科，竟然有这样多的人，而且瞧上去也都充满希望。

但她还是踏上了回来亚什基的火车。

来亚什基也变了。镇上多了她不认识的脸，广场中央的雕塑换了模样。学校，学校还是老样子，孩子们在小操场上做游戏。镇民们的问候略显局促，没人提起她此前匆匆告别的事，似乎这并不重要，所有人随时都能离开，也随时能回到此地。

雪落上了她的头发和领口，让她的手指和鼻尖发红，这是真正的雪。

# 孩子的创造
由不同的小手塑造，分享着同一种纯然的希冀。
圆圈舞
大小不一的雪人在操场上围成了圈，孩子们贴心地将它们的树枝手臂两两绑在一起，让这群雪朋友可以手拉着手。他们各自专注地装饰自己的小小造物，同样贴得离彼此很近，每当谁的胳膊撞上谁的肩膀，就会响起快活的笑声。

最左边的小姑娘低下头，辫子像太阳下的草垛儿一样翘起，闪闪发亮。她把向日葵花瓣仔细地贴在雪人头顶。旁边，两个男孩子试图把雪人胖胖的身躯变得再宽些，再结实些，好挨过更长时间，使它不太快融化——这样维拉老师能多跟它们打几次招呼。

属于小个子男孩儿的雪人是一只小熊，尽管需要依靠他的介绍，才能区分出脑袋、爪子、熊背上趴着的北极狐。狐狸通常可没有兔子的长耳朵，但他会让它拥有这个，好听清妈妈的脚步、维拉的脚步、好朋友们的脚步。现在是最后的工序了，他把找到的矿石仔细地安放在狐狸胸口，就像他把爸爸留下的漂亮石头挂在胸前。

“好啦，雪人同志们！你们要好好留在这里，和维拉老师一块儿跳圆圈舞，别乱跑，别吵闹，听她的话……”

# 工人们
那些令来亚什基矗立在雪原上的人，那些从未期待过留下名字的人。
温暖的人群
在长桌与长桌之间有这样一群人。

被围在中间的那个，维持着她惯常的礼貌、友善的笑容，如果不将目光挪向桌面那只不知如何安放的手，就会错过这层礼貌外的情绪——不可置信——是的，这位年轻的学者震惊于她所专注的理论如今再次备受瞩目，在外面多数地方把理线学扫进旧纸堆之后，它在来亚什基却有了容身之所。

至于她周围的人，多数还身着工作服，煤灰爬上了他们的领口和裤腿。他们自己也清楚这点，因此和学者保持着距离，直到她伸出手。这群临时凑成的老师和学生彼此握手，决心花时间了解一门没落的学科。最热衷于劳动的人们也是最认真的实用主义者，理线学在小镇的重建工作里起了怎样重要的作用啊！最初，在走进研究所的时候，学者正怀抱着类似的期望。理线学是万能的吗？不。但它或许能在某个时间带来改变。

也许是发现煤矿的那一刻，也许就是这一刻。
