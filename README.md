# 重返未来：1999 剧情 Wiki

> 《重返未来：1999》（Reverse: 1999）同人剧情知识库，供同人创作参考使用。

---

## 项目简介

本项目是对《重返未来：1999》游戏官方剧情文本的系统化整理与归档，涵盖：

- **主线故事**（序幕至第十三章，共 14 章，其中第十三章暂未录入）
- **活动章节**（限时独立剧情 + 章节配套活动，共 27 个）
- **角色页面**（172 个角色，含基础信息与剧情经历）
- **小径碎片**（28 个小径补充片段，已整合至各章节页面；另有独立世界观页面记录小径中???角色分析）
- **世界观设定**（暴雨机制、神秘学体系、时间线等）

Wiki 的主要用途是**支持同人创作**，特别关注角色说话风格、人物关系、世界观细节与剧情时间线。

## Wiki 使用方式

### 用 Obsidian 打开

将 `wiki/` 目录作为 Obsidian Vault 打开，即可使用内链导航、标签检索等功能。文件均以数字前缀排序，在文件浏览器中按章节/时间顺序显示。

### 快速查找

- **全局索引**：`wiki/index.md` — 主线/活动两栏导航，含时间线与格言
- **角色页面**：`wiki/角色/<角色名>.md` — 基础信息、剧情经历、角色关系
- **主线摘要**：`wiki/剧情概要/00_序幕_此即明日.md` 起，按章节顺序
- **活动摘要**：`wiki/剧情概要/活动章节/01_雷米特杯失窃案.md` 起，按开放时间顺序

## 技术说明

### 原始素材获取

`tools/` 目录下的 Puppeteer 脚本用于从[灰机wiki（huijiwiki.com）](https://res1999.huijiwiki.com/)爬取游戏剧情页面，提取并保存为 Markdown 格式至 `raw/`。

```bash
cd tools
node scrape-custom.js   # 按自定义 URL 列表爬取
node extract-content.js # 从已下载页面提取正文
```

### Wiki 维护

Wiki 由 Claude（LLM）通过 Claude Code 进行批量 Ingest 与维护，每次处理一批原始文件（通常为一个完整活动或章节），写入 `wiki/` 对应页面，并更新 `wiki/index.md`。

具体规范见 `schema/CLAUDE.md`。
