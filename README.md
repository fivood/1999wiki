# 重返未来：1999 剧情 Wiki

> 《重返未来：1999》（Reverse: 1999）同人剧情知识库，供同人创作参考使用。

---

## 项目简介

本项目是对《重返未来：1999》游戏官方剧情文本的系统化整理与归档，涵盖：

- **主线故事**（序章至第十二章及以后）
- **活动章节**（限时独立剧情，共18个活动）
- **角色剧情**（个人故事）
- **世界观设定**（暴雨机制、神秘学体系等）

Wiki 的主要用途是**支持同人创作**，特别关注角色说话风格、人物关系、世界观细节与剧情时间线。

---

## 目录结构

```
1999story/
├── raw/                    # 原始素材（从游戏 Wiki 提取，只读）
│   ├── 主线故事/            # 主线章节文本（1ST-01 等）
│   ├── 活动剧情/            # 活动事件文本（TTSC-01 等）
│   ├── 角色剧情/            # 角色个人故事
│   ├── 角色/               # 所有角色的基础资料页
│   ├── 小径/               # 小径故事内容与补充设定
│   ├── 黄昏的音序/          # Roguelike 剧情（以塞梅尔维斯为主角）
│   ├── 第三扇门/            # 探索解密类世界观碎片
│   ├── 轶事/               # 短篇故事
│   └── story-links.json    # 所有页面链接索引
│
├── wiki/                   # LLM 生成与维护的知识库（可供 Obsidian 打开）
│   ├── index.md            # 全局索引（记录所有批次的 Ingest 历史）
│   ├── log.md              # 操作日志
│   ├── 角色/               # 角色百科页面（含活动章节经历）
│   ├── 世界观/              # 暴雨、神秘学等概念
│   ├── 地点/               # 重要地点
│   ├── 组织/               # 基金会、重塑之手等
│   ├── 主题/               # 主题分析
│   └── 剧情概要/            # 剧情梳理（主线 + 活动章节）
│
├── schema/
│   └── CLAUDE.md           # Wiki 编写规范（供 LLM 参考）
│
├── tools/                  # 爬虫脚本（用于从游戏 Wiki 提取原始素材）
│   ├── scrape-*.js         # Puppeteer 爬虫脚本
│   ├── extract-*.js        # 内容提取脚本
│   └── *.png               # 无法用文字提取的图片内容（手动参考）
│
└── README.md               # 本文件
```

## Wiki 使用方式

### 用 Obsidian 打开

将 `wiki/` 目录作为 Obsidian Vault 打开，即可使用内链导航、标签检索等功能。

### 快速查找

- **全局索引**：`wiki/index.md` 列出所有已 Ingest 的内容批次与对应角色
- **角色页面**：`wiki/角色/<角色名>.md`，包含基础信息、剧情经历与角色关系
- **活动章节**：`wiki/剧情概要/活动章节/<活动名>.md`，完整的分幕剧情概要

## 技术说明

### 原始素材获取

`tools/` 目录下的 Puppeteer 脚本用于从[灰机wiki（huijiwiki.com）](https://res1999.huijiwiki.com/)爬取游戏剧情页面，提取并保存为 Markdown 格式至 `raw/`。

```bash
cd tools
node scrape-custom.js   # 按自定义 URL 列表爬取
node extract-content.js # 从已下载页面提取正文
```

### Wiki 维护

Wiki 由 Claude（LLM）通过 Claude Code 进行批量 Ingest 与维护，每次处理一批原始文件（通常为一个完整活动或章节），写入 `wiki/` 对应页面，并更新 `wiki/index.md`。

具体规范见 `schema/CLAUDE.md`。
